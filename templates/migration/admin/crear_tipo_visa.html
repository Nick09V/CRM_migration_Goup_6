¿Qué es un Tipo de Visa?

Los tipos de visa definen las categorías de trámites migratorios que los solicitantes pueden solicitar. Cada tipo de visa puede tener requisitos específicos asociados.

Recomendaciones
  • Código: Use un identificador corto y sin espacios (ej: "diplomatica", "negocios")
  • Nombre: Use un nombre descriptivo que será visible para los usuarios
  • Descripción: Opcional, pero útil para documentar el propósito de la visa
Ejemplos de Tipos de Visa
  • Diplomática: Para personal diplomático y sus familias
  • Negocios: Para viajes de negocios cortos
  • Médica: Para tratamientos médicos en el extranjero
